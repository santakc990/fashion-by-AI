Smart Wardrobe AI 👕✨

Your personal AI stylist that actually knows what’s in your closet and helps you dress better every day.

No more “I have nothing to wear” — your wardrobe does the thinking for you.

🎯 Features

* 👕 Add your clothes, shoes, bags, and accessories
* 🧠 Smart wardrobe memory (it remembers everything you own)
* 🎨 AI outfit recommendations based on mood + weather
* 🌦️ Weather-aware styling (hot, cold, rainy fits)
* 😎 Mood-based outfits (lazy, confident, sporty, etc.)

⚡ Tech Stack

* React + TypeScript ⚛️
* Tailwind CSS 🎨
* Framer Motion ✨ (smooth outfit transitions)
* Future-ready for AI integration 🤖

🚀 Future Upgrades

* 🔐 Firebase login + cloud wardrobe sync
* 📸 Upload outfit photos like Instagram for clothes
* 🤖 Real AI stylist (ChatGPT-powered fashion brain)
* 🌤️ Live weather API outfit suggestions
* 🔥 Trend detection (what’s in style right now)
* 🪞 Virtual outfit preview

🧑‍💻 Getting Started

```bash
npm install
npm run dev
```
💡 Idea

Turn your closet into a smart system that thinks like a stylist, so getting dressed feels fast, easy, and actually fun.
